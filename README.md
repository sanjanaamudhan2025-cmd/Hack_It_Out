CareMetrics Insight: Administrative Analytics Dashboard
Team Name: Hack_it_out
Event: CodeChef VIT Round 1
Project Overview
CareMetrics Insight is a centralized Administrative Analytics Dashboard designed to help hospital administrators monitor operational activities efficiently. The system transforms raw hospital data into structured visual insights, such as workload distribution and patient trends, to support faster, data-driven decision-making.

The Problem
Hospital systems are often designed for basic record-keeping rather than analysis. Administrators frequently struggle with:

Data Fragmentation: Operational data is distributed across multiple systems.

Manual Reporting: Dependence on spreadsheets leads to time-consuming and inefficient monitoring.

Lack of Visualization: Identifying workload imbalances or performance trends is difficult without proper tools.

The Solution
A three-layer architecture (Frontend, Backend, Database) that provides a centralized platform for monitoring hospital operations.

Frontend: Built with React and Tailwind CSS for a responsive, user-friendly interface.

Backend: Powered by Node.js to process data and calculate operational statistics.

Visualizations: Uses Chart.js to generate interactive bar and pie charts.

Technology Stack
Core: React, TypeScript, Vite

Styling: Tailwind CSS

Charts: Chart.js

Database Integration: Structured for MySQL/Supabase

Key Features
Patient Analytics: Identify busy hospital areas and visit trends.

Workload Analysis: Monitor how work is distributed across different departments.

Resolution Tracking: Analyze the time taken to complete cases and identify delays.

Responsive UI: Optimized layout featuring a statistics header, interactive charts, and detailed data tables.

System Architecture
User → Frontend (React/Vite) → Backend (Node.js) → Database
