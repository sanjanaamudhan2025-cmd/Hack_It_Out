CareMetrics Insight
Project Overview

CareMetrics Insight is a web-based application designed to manage and visualize healthcare-related data in an organized and user-friendly way. The system provides an interactive interface for storing, viewing, and analyzing information efficiently.

The application focuses on simplicity, performance, and clean user interface design.

Features

User-friendly interface

Fast and responsive design

Data management system

Interactive components

Clean dashboard layout

Mobile-friendly design

Technologies Used

This project is developed using:

React.js

TypeScript

Vite

Tailwind CSS

shadcn-ui

Installation

Follow these steps to run the project locally.

Step 1 – Clone Repository
git clone https://github.com/lisamehta0791/caremetrics-insight
Step 2 – Open Project Folder
cd caremetrics-insight
Step 3 – Install Dependencies
npm install
Step 4 – Run Project
npm run dev

After running the command, open the local server link shown in the terminal.

Project Structure

Main folders used in the project:

src/
 ├── components
 ├── pages
 ├── styles
 ├── assets
 └── utils

components → Reusable UI components

pages → Application pages

styles → Styling files

assets → Images and static files

utils → Helper functions

Build for Production

To create a production build:

npm run build
